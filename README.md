# EAE1106 - Métodos Computacionais para Economia
Repositório com o material da disciplina **EAE1106 - Métodos Computacionais para Economia**.

Este projeto reúne notas de aula, códigos, exemplos aplicados e outros recursos utilizados ao longo do curso. O objetivo é fornecer uma base prática para a utilização de ferramentas computacionais em problemas econômicos.
